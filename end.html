句尾功能模組

理論全活用 × 實用封印
V = 任意動詞 Vる = 辭書形 Vて = て形
㊤ 高頻 ㊥ 限定 ㊦ 封印
Module 1｜Vている（狀態）
狀態・現況（進行／定著）
世界「現在就是這樣」的描述模組。
Forms
  • Vている ㊤
    現在正處於這個狀態
  • Vていない ㊤
    目前並不是這樣
  • Vていた ㊤
    當時處於那個狀態
  • Vていなかった ㊥
    當時其實不是那樣
  • Vています ㊤
    目前是這個狀態（禮貌）
  • Vていません ㊤
    目前不是這個狀態（禮貌）
  • Vていました ㊤
    當時是那個狀態（正式回顧）
  • Vていませんでした ㊥
    當時並非如此（正式澄清）
  • Vているんです ㊤
    我現在的狀況就是這樣（說明）
  • Vていなかったんです ㊥
    其實當時並沒有那樣（說明）
  • Vているわけです ㊥
    結論來說，現在是這個狀態
  • Vていなかったわけです ㊦
    ❌（狀態描述與結論判斷衝突）
Note

封印核心：Vている 本身是「世界描述中」，太早接「結論（わけだ）」容易語感拒絕。

Module 1-補｜Vてある（結果狀態）
目的性結果 → 現在維持
有人做過 → 現在保持在完成狀態。
Forms
  • Vてある ㊤
    已經做好並保持著
  • Vてあった ㊤
    當時已經處理完成
  • Vてあります ㊤
    已準備完成（文件／現況）
  • Vてありました ㊥
    當時已完成（正式回報）
  • Vてあるんです ㊥
    所以現在是完成狀態（說明）
  • Vてあるわけです ㊦
    ❌（結果本身不需要再下結論）
Note

主語通常是「結果」而非人；用於說明書、標示、現況描述很高頻。

Module 2｜Vてくる（過去→現在）
變化一路逼近「現在」
過去累積／趨勢一路走到此刻。
Forms
  • Vてくる ㊤
    狀況正一路朝現在靠近
  • Vてきた ㊤
    已經一路走到現在
  • Vてこなかった ㊤
    過去一直都沒有
  • Vてきます ㊤
    將一路延續到現在（禮貌）
  • Vてきました ㊤
    已累積至今（回報）
  • Vてきたんです ㊤
    現在會這樣是有來歷的
  • Vてこなかったんです ㊥
    其實一路以來都沒有（對比／反駁）
  • Vてきたわけです ㊥
    這樣一路走來是有原因的（論述）
  • Vてこなかったわけです ㊦
    ❌（否定累積＋結論過重）
Note

常與 Vている 合體 → Vてきている（現在狀態＝累積而來）。

Module 2｜Vていく（現在→未來）
從「現在」開始往後延展
此刻起的推進／延續宣告。
Forms
  • Vていく ㊤
    從現在開始往後推進
  • Vていった ㊤
    當時開始一路延續
  • Vていきます ㊤
    接下來會這樣做（禮貌）
  • Vていくつもりだ ㊤
    心裡已決定要這樣走
  • Vていく予定です ㊤
    已經排定會這樣進行
  • Vていくんです ㊥
    方向確定是往這邊走（強調）
  • Vていくわけです ㊥
    說明一下接下來的走向
  • Vていかないわけです ㊦
    ❌（否定方向＋結論衝突）
Note

Vていく 後面接 予定／つもり 最自然；也常接 べき／必要がある（評論）。

Module 2-核心｜Vてきている（累積狀態）
過去累積 → 現況（分析引擎）
為什麼「現在」會是這樣（非常高頻）。
Forms
  • Vてきている ㊤
    現在的狀態是長期累積的結果
  • Vてきていた ㊤
    當時已經累積到那一步
  • Vてきています ㊤
    目前呈現出累積至今的狀態
  • Vてきているんです ㊤
    所以現在才會變成這樣
  • Vてきていない ㊥
    本來該累積但實際沒有
  • Vてきていなかった ㊥
    當時並未形成應有的累積（評論）
  • Vてきているわけです ㊥
    綜合來看就是這樣演變（論述）
  • Vてきていなかったわけです ㊦
    ❌（多層否定＋結論過載）
Note

新聞、評論、分析對話的主引擎：看到就先抓「來歷→現況」這條線。

Module 3｜〜ようになる（門檻）
跨過臨界點（不能→能／不會→會）
能力／習慣的「跨線」描述。
Forms
  • Vるようになる ㊤
    從不能到能
  • Vるようになった ㊤
    已經跨過那條門檻
  • Vるようになります ㊤
    將會變得能做到
  • Vるようになってきている ㊤
    正逐漸達到能做的狀態
  • Vるようになっていく ㊤
    之後會越來越能做到
  • Vるようにならなかった ㊥
    最終仍未跨過門檻
  • Vるようにならなかったんです ㊥
    結果是沒能做到（說明）
  • Vるようにならなかったわけです ㊦
    ❌（門檻失敗不適合再下結論）
Note

常與世界線合體：〜ようになってきている／〜ようになっていく。

Module 4｜〜てしまう（完結）
完結／不可逆＋情緒收束
完成、收尾、甚至『糟了』都包。
Forms
  • Vてしまう ㊤
    做完了／事情收尾
  • Vてしまった ㊤
    已成定局
  • Vてしまいます ㊤
    即將完成（禮貌）
  • Vてしまったんです ㊤
    已經發生，帶情緒（說明）
  • Vてしまっている ㊤
    已完成並卡在結果狀態
  • Vてしまわなかった ㊥
    結果沒有走到完結（對比）
  • Vてしまわなかったわけです ㊦
    ❌（否定完結＋結論冗餘）
Note

你看到 〜てしまっている，多半就是『完了而且狀態還在』。

Module 5｜〜ことになる（制度／推導）
不是我決定，是制度／邏輯推導
結果歸結（規則、安排、推論）。
Forms
  • Vることになる ㊤
    推導結果是要那樣做
  • Vることになった ㊤
    已確定會那樣做
  • Vることになります ㊤
    規則／安排上會那樣做（禮貌）
  • Vることになっている ㊤
    本來制度就是如此
  • Vることになったんです ㊤
    結果最後就變成那樣（說明）
  • Vることにならなかった ㊥
    最終沒有變成那樣（取消／未成立）
  • Vることにならなかったわけです ㊦
    ❌（制度結果不需再下結論）
Note

〜ことになっている 是『制度態』：規則／流程／慣例。

Module 6｜意圖（つもり／予定）
意圖／計畫（人主觀 vs 客觀安排）
內心決定與行程安排的句尾封口。
Forms
  • Vるつもりだ ㊤
    我打算那樣做
  • Vるつもりです ㊤
    我打算那樣做（禮貌）
  • Vる予定です ㊤
    安排上已排定會那樣做
  • Vるつもりだった ㊤
    當時本來打算那樣做
  • Vるつもりではなかった ㊥
    原本並沒有這個打算（澄清）
  • Vるつもりではなかったわけです ㊦
    ❌（意圖否定＋結論過載）
Note

Vていく＋予定／つもり 是母語者超自然組合。
