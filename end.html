句尾功能模組

理論全活用 × 實用封印
㊤ 高頻 ㊥ 限定 ㊦ 封印
Module 1｜Vている（狀態）
狀態・現況（進行／定著）
世界「現在就是這樣」的描述模組。
Forms
  • している ㊤
    現在正處於這個狀態
  • していない ㊤
    目前並不是這樣
  • していた ㊤
    當時處於那個狀態
  • していなかった ㊥
    當時其實不是那樣
  • しています ㊤
    目前是這個狀態（禮貌）
  • していません ㊤
    目前不是這個狀態（禮貌）
  • していました ㊤
    當時是那個狀態（正式回顧）
  • していませんでした ㊥
    當時並非如此（正式澄清）
  • しているんです ㊤
    我現在的狀況就是這樣（說明）
  • していなかったんです ㊥
    其實當時並沒有那樣（說明）
  • しているわけです ㊥
    結論來說，現在是這個狀態
  • していなかったわけです ㊦
    ❌（狀態描述與結論判斷衝突）
Note

封印核心：Vている 本身是「世界描述中」，太早接「結論（わけだ）」容易語感拒絕。

Module 1-補｜Vてある（結果狀態）
目的性結果 → 現在維持
有人做過 → 現在保持在完成狀態。
Forms
  • してある ㊤
    已經做好並保持著
  • してあった ㊤
    當時已經處理完成
  • してあります ㊤
    已準備完成（文件／現況）
  • してありました ㊥
    當時已完成（正式回報）
  • してあるんです ㊥
    所以現在是完成狀態（說明）
  • してあるわけです ㊦
    ❌（結果本身不需要再下結論）
Note

主語通常是「結果」而非人；用於說明書、標示、現況描述很高頻。

Module 2｜Vてくる（過去→現在）
變化一路逼近「現在」
過去累積／趨勢一路走到此刻。
Forms
  • してくる ㊤
    狀況正一路朝現在靠近
  • してきた ㊤
    已經一路走到現在
  • してこなかった ㊤
    過去一直都沒有
  • してきます ㊤
    將一路延續到現在（禮貌）
  • してきました ㊤
    已累積至今（回報）
  • してきたんです ㊤
    現在會這樣是有來歷的
  • してこなかったんです ㊥
    其實一路以來都沒有（對比／反駁）
  • してきたわけです ㊥
    這樣一路走來是有原因的（論述）
  • してこなかったわけです ㊦
    ❌（否定累積＋結論過重）
Note

常與 Vている 合體 → してきている（現在狀態＝累積而來）。

Module 2｜Vていく（現在→未來）
從「現在」開始往後延展
此刻起的推進／延續宣告。
Forms
  • していく ㊤
    從現在開始往後推進
  • していった ㊤
    當時開始一路延續
  • していきます ㊤
    接下來會這樣做（禮貌）
  • していくつもりだ ㊤
    心裡已決定要這樣走
  • していく予定です ㊤
    已經排定會這樣進行
  • していくんです ㊥
    方向確定是往這邊走（強調）
  • していくわけです ㊥
    說明一下接下來的走向
  • していかないわけです ㊦
    ❌（否定方向＋結論衝突）
Note

Vていく 後面接 予定／つもり 最自然；也常接 べき／必要がある（評論）。

Module 2-核心｜Vてきている（累積狀態）
過去累積 → 現況（分析引擎）
為什麼「現在」會是這樣（非常高頻）。
Forms
  • してきている ㊤
    現在的狀態是長期累積的結果
  • してきていた ㊤
    當時已經累積到那一步
  • してきています ㊤
    目前呈現出累積至今的狀態
  • してきているんです ㊤
    所以現在才會變成這樣
  • してきていない ㊥
    本來該累積但實際沒有
  • してきていなかった ㊥
    當時並未形成應有的累積（評論）
  • してきているわけです ㊥
    綜合來看就是這樣演變（論述）
  • してきていなかったわけです ㊦
    ❌（多層否定＋結論過載）
Note

新聞、評論、分析對話的主引擎：看到就先抓「來歷→現況」這條線。

Module 3｜〜ようになる（門檻）
跨過臨界點（不能→能／不會→會）
能力／習慣的「跨線」描述。
Forms
  • できるようになる ㊤
    從不能到能
  • できるようになった ㊤
    已經跨過那條門檻
  • できるようになります ㊤
    將會變得能做到
  • できるようになってきている ㊤
    正逐漸達到能做的狀態
  • できるようになっていく ㊤
    之後會越來越能做到
  • できるようにならなかった ㊥
    最終仍未跨過門檻
  • できるようにならなかったんです ㊥
    結果是沒能做到（說明）
  • できるようにならなかったわけです ㊦
    ❌（門檻失敗不適合再下結論）
Note

常與世界線合體：〜ようになってきている／〜ようになっていく。

Module 4｜〜てしまう（完結）
完結／不可逆＋情緒收束
完成、收尾、甚至『糟了』都包。
Forms
  • してしまう ㊤
    做完了／事情收尾
  • してしまった ㊤
    已成定局
  • してしまいます ㊤
    即將完成（禮貌）
  • してしまったんです ㊤
    已經發生，帶情緒（說明）
  • してしまっている ㊤
    已完成並卡在結果狀態
  • してしまわなかった ㊥
    結果沒有走到完結（對比）
  • してしまわなかったわけです ㊦
    ❌（否定完結＋結論冗餘）
Note

你看到 〜てしまっている，多半就是『完了而且狀態還在』。

Module 5｜〜ことになる（制度／推導）
不是我決定，是制度／邏輯推導
結果歸結（規則、安排、推論）。
Forms
  • 行くことになる ㊤
    推導結果是要去
  • 行くことになった ㊤
    已確定要去
  • 行くことになります ㊤
    規則上會這樣（禮貌）
  • 行くことになっている ㊤
    本來制度就是如此
  • 行くことになったんです ㊤
    結果最後變成這樣（說明）
  • 行くことにならなかった ㊥
    最終沒有成行（取消）
  • 行くことにならなかったわけです ㊦
    ❌（制度結果不需再下結論）
Note

〜ことになっている 是『制度態』：規則／流程／慣例。

Module 6｜意圖（つもり／予定）
意圖／計畫（人主觀 vs 客觀安排）
內心決定與行程安排的句尾封口。
Forms
  • 行くつもりだ ㊤
    我打算要去
  • 行くつもりです ㊤
    我打算要去（禮貌）
  • 行く予定です ㊤
    行程上已排定
  • 行くつもりだった ㊤
    當時本來打算去
  • 行くつもりではなかった ㊥
    原本並沒有這個打算（澄清）
  • 行くつもりではなかったわけです ㊦
    ❌（意圖否定＋結論過載）
Note

Vていく＋予定／つもり 是母語者超自然組合。
