形容詞活用

只收「實際會講、實際會聽到」的版本
① い形容詞（例：高い）
核心：語幹 + い ／ 口語丁寧：原形 + です
不用背硬式敬語，日常就靠「です／でした」。
常用活用（口語安全）
  • 基本：高い → 高いです
  • 否定：高くない → 高くないです
  • 過去：高かった → 高かったです
  • 過去否定：高くなかった → 高くなかったです
  • て形：高くて
  • 副詞：高く
常見接續
  • 推量：高いでしょう
  • 條件：高ければ
備註：高くありません／高くありませんでした 偏硬、偏櫃檯/文件感，日常對話可不必用。
② な形容詞（例：静か）
核心：語幹 + な／だ ／ 口語丁寧：〜です 系
判斷只看「後面接什麼」：名詞？句尾？接續？
「な／だ」使用條件（一句判斷法）

後面是名詞 → 用「な」

  • 静かな部屋
  • 便利なアプリ
  • 大事な話

句子結尾（斷句）→ 用「だ／です」

  • 部屋は静かだ
  • それは大事です

要接續／變形 → 兩個都不用

  • 静かで助かる
  • 静かに話す
  • 静かなら行く
常用活用（口語安全）
  • 基本：静かだ → 静かです
  • 否定：静かじゃない → 静かじゃないです
  • 過去：静かだった → 静かでした
  • 過去否定：静かじゃなかった → 静かじゃなかったです
  • て形：静かで
  • 副詞：静かに
備註：ではありません／ではありませんでした 偏正式，日常口語多用 じゃないです 系。
③ 不規則：いい（良い）
只有基本形是 いい，其餘全部用 よ
這個不規則值得單獨背：背一行就好。
口語安全形
  • 基本：いいです
  • 否定：よくないです
  • 過去：よかったです
  • 過去否定：よくなかったです
  • て形：よくて
  • 副詞：よく
常見接續
  • 推量：いいでしょう
  • 條件：よければ
避免：いくない／いかった（常見誤造）。
